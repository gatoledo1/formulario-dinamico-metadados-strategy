# Skill para IA (Knowledge Skill)

**dynamic_form_architecture_skill.md**

### Objetivo:

- Ensinar um conceito.
- Servir como documentação especializada.
- Explicar arquitetura, componentes, responsabilidades e exemplos.

**É equivalente a um:**
```
Guia técnico
+
Documentação arquitetural
+
Material de aprendizado
```

**Uma IA lê isso para entender:**
```
o que é a arquitetura;
como ela funciona;
quais componentes existem;
como as partes se relacionam.
```
Ela é mais parecida com um README avançado.

# Skill para Agente (Behavior Skill)

**dynamic_form_architecture_agent_skill.md**

### Objetivo:

- Controlar comportamento.
- Guiar tomada de decisão.
- Definir regras operacionais.

**É equivalente a um:**
```
Manual operacional
+
Prompt de sistema
+
Conjunto de regras
```

**Um agente lê isso para saber:**
```
como pensar;
quais decisões tomar;
o que é permitido;
o que é proibido;
como revisar o próprio trabalho.
```

### Exemplo:

**O primeiro documento diz:**

- O Renderer não deve conhecer regras de negócio.

**O segundo documento diz:**

- Antes de gerar código, verifique se a regra pertence ao Renderer. Se pertencer ao domínio, mova para o Metadado.


O primeiro explica.

O segundo instrui.
